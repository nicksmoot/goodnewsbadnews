# Roadmap

## Milestone 1 — Working MVP

- Build Next.js application shell and routes.
- Add reusable civic components.
- Add Supabase schema and signal submission endpoint.
- Add moderation approval endpoint.
- Add Stripe webhook placeholder and product definitions.
- Add Resend Saturday digest endpoint.

## Milestone 2 — Production moderation

- Add authenticated moderator dashboard.
- Add AI review queue with explicit safety categories.
- Add correction request and right-of-reply forms.
- Add audit log for status changes.

## Milestone 3 — Spokane launch

- Seed Spokane neighborhoods.
- Invite founding contributors.
- Publish first Saturday briefing.
- Add newsroom operating procedures.

## Milestone 4 — Multi-city platform

- Add city provisioning workflow.
- Add city-specific subscriptions and editorial roles.
- Add public record integrations.
